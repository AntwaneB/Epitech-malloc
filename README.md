Malloc
===================

Ce dépôt contient les sources du Malloc pour le module Système Unix de 2e année à Epitech.
Il a été réalisé à Strasbourg en 2015.

----------

Fonctionnalités
-------------

> - malloc
> - free
> - realloc
> - statistiques d'allocation et de free
